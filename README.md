#  MathOff

MathOff é um aplicativo educativo e divertido para treinar habilidades matemáticas de forma interativa e gamificada. Desenvolvido com React Native, o app permite aos usuários jogar desafios, acumular pontuações e acompanhar estatísticas de desempenho.

---

## Funcionalidades

- ✅ Jogue desafios com operações matemáticas básicas: **adição, subtração, multiplicação e divisão**.
- 🧠 Registra o **histórico de pontuações** e apresenta estatísticas de desempenho.
- 🎯 Diferentes **modos e níveis de dificuldade**.
- 📊 Visualização de **estatísticas** com total de jogos e maior pontuação.
- 🗑️ Opções para limpar os **cards visíveis** ou **todo o histórico**.
- 🎨 Interface animada com **gradientes e transições suaves**.
